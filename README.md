# Проект: Место

###В проекте мы реализовали(использовали):

- Формы
- Изображения
- Отступы
- Шрифты
- и т.д

### Технологии

- Методология БЭМ
- Grid
- Адаптивная верстка
- Figma
- JavaScript
- DOM
